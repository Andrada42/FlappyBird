# Flappy Bird

video game similar to Flappy Bird and a showcase site

## How to run the game

Download **Game/Build Final** directory  

```cmd
cd "path\to\Game\Build Final"
python -m http.server 8000
```

Open a browser and type:
http://localhost:8000

## About the site
Open **index.html** file  
The source code of this project is complete and functional. However, due to the expiration of the hosting service, some external assets (images/resources) might not load properly in the live preview.
